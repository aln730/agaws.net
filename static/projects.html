┗━ gatekeeper-pam

        A Linux PAM module enabling NFC-based authentication.

   ┣━ Centralized identity: keys are tied to LDAP uids, so access control is managed in one place 
   ┃  across all machines.
   ┣━ Privilege separation: NFC hardware and realm keys live in a dedicated daemon; the PAM module 
   ┃  is a thin Unix socket client.
   ┗━ Secure: backed by DESFire AES-encrypted NFC authentication.
